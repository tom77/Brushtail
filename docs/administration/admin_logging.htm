Logging user activity

User logins



Only the administrator user can view user login statistics.

All user visits are logged. These records can be deleted automatically after a specified number of months, this is set under the "Database cleanup".





To see visits by day click on the month name.





To see a list of visits for the day click on the day name.





The username, ip address and time of visit is displayed for each visit.





Failed user Logins



The failed user logins statistics are the same as the user logins above except:



Content updates



When viewing content through a content management view, the username, date and ip address of when the content was last created/updated is displayed below paragraph title.


PC, Room & equipment Bookings



The date, intranet username and IP address are recorded by the room bookings module. This is visible at the bottom of the edit view of a booking. Username and time of booking visible to edit users with edit privileges. Administrator can also see IP address of the booking. details.



Calendar bookings



The intranet username and IP address of the booking is visible to the intranet administrator in grey text.


PC bookings



The intranet username, IP address and time of the booking is visible to the intranet administrator. This is visible at the bottom of the screen when editing a booking.
