Brushtail Administrator's Guide

Branch Locations

Introduction



Only the administrator user can edit Branch Locations.

Creating branch locations:
  1. Result in branches being available for PC bookings
  2. Branches will be listed in a list of locations in the IT Problems form.
  3. Branches will also appear in a list when adding hardware to the hardware list in the IT Jobsheet module.
Click on "Branch locations".





You can now create and edit branch locations.
