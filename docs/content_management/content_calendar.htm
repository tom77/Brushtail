Brushtail Content Management Guide

 

Introduction

To create a calendar to edit.

  1. The administrator must create a calendar module.
  2. The administrator must define one or more event types.
  3. The administrator must allocate one or more event types to the calendar module.
  4. Give an intranet user edit permissions so that they can access the calendar module through content management.

 

Calendar events

To edit a calendar you will need to to choose the calendar module from the module listing on the Content Management page.

Add event



Click on the calendar cell of the day on which you want to create the event.





You will then have a choice of event types that the calendar allows. The number of event types listed is dependant upon the number of event types defined by the administrator. It is also dependant upon which types this calendar can use. This is set by the administrator.



After choosing the event type, enter details of the event in the add event form. The number of fields on this form will depend upon how the event type was defined by the administrator.

 

Edit event

To update, cancel or delete an event, click on the event name.





The edit event form allows you to change event details.





To cancel an event, change the status to "cancelled". A cancelled event will display on the calendar as cancelled. Selecting delete will remove the event and any related bookings completely from the calendar.





Recurrent events

An event type may be defined as capable of recurrent bookings. (see the "Calendars" page of the administrator manual on how to do this). This allows you to to create multiple events in one go.

To make an event recurrent, select "yes", choose the recurrence period and number of occurrences.

Events that are created recurrently may be later edited individually or as a group.

Create event templates

If you need to create a number of events that have similar details you can reduce the amount of manual input by using event templates.





Click on "Edit event templates". A list of any existing templates will appear that can be edited. To create a new event template click on "Add template".





Choose an event type for the template.





The next form is the same as creating an event except that the form has a button "add template" instead of "Add event".





Event tags

You can add tags that will help with keyword searching for events. Tags are not displayed to the end user. A collection of tags may be stored for reuse.

Using event templates

 

Event templates is a fast way of adding repetitious input. Follow the usual steps for creating an event. When you get to the Add Event form, there will be a list of available event templates for this event type.







If you click on one of the event templates it will populate the Add Event form with the template details. Change any details as necessary and then click on "Add event".


Statistics

 

If you calendar has event types that take bookings, you can access booking statistics.







These consist of monthly totals for each event type.
