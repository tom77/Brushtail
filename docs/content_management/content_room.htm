Room & Equipment Bookings

Introduction

The Room & equipment bookings system can have a number of uses. These include meeting/training/community room bookings, equipment bookings or even display case bookings. To edit a Room booking module click on it in the Content management menu.

 

When you click on "Room bookings" on the Content Management page. You will then see an "Add" icon and below this a list of existing resources that can be edited.



Adding a resource



The "Add resource" form allows you to define what fields will appear on the end user booking form.





Custom fields
You can define custom fields for the room bookings form.

Print
Booking receipt can be enabled. Options include full page receipts or transaction printer receipts.

Fee
If room bookings can have a fee, the bookings will be colour coded.
Orange = unpaid
Light green = partially paid
Dark green = fully paid


Multiple day bookings
You can specify whether bookings for this resource can span more than one day.

Email notification of booking
If this parameter is set to "Yes" and an email address specified, then an email notification will be sent every time a booking is made.

Maximum recurrence
Recurrent bookings may be enabled for a room.

Checkout enabled.
Bookings may be given a "checkedout" status. This will display red on the calendar. This indicates that this booking currently has room (or equipment). May be of use if a room has a key that needs to be returned.
